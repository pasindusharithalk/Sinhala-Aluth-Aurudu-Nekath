# Sinhala Aluth Avurudu 2026 - Nekath Tracker 🌞

A beautifully designed, interactive web application I built to track the auspicious times (Nekath) for the Sinhala and Tamil New Year 2026. 

I created this project to help people easily keep track of important cultural rituals. It features precise real-time countdowns, browser push notifications, and a clean, culturally inspired user interface.

## ✨ Features

* **Real-Time Countdown:** Accurately counts down to the next upcoming auspicious time based on Sri Lanka Standard Time (UTC+5:30).
* **Smart Push Notifications:** Sends browser notifications 15, 10, and 5 minutes before a Nekath, as well as an exact-time alert with detailed instructions in Sinhala and English.
* **On-Screen Alerts:** A beautiful modal with confetti animations pops up when an auspicious time arrives, ensuring you don't miss it even if push notifications are disabled.
* **Progressive Web App (PWA):** Users can install the app directly to their mobile home screens for a native app-like experience.
* **Responsive Design:** Fully optimized for both desktop and mobile devices.

## 🛠️ Technologies Used

I built this project using a modern but lightweight frontend stack:
* **HTML5 & CSS3** (Custom styling with Tailwind CSS utility classes)
* **Vanilla JavaScript** (No heavy frameworks, ensuring fast load times)
* **Vite** (For fast development and optimized production builds)
* **Service Workers** (For PWA capabilities and offline caching)

## 🚀 How to Run Locally

If you want to run this project on your own machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pasindusharithalk/Sinhala-Aluth-Aurudu-Nekath.git
   cd YOUR_REPO_NAME
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.

## 📦 Building for Production

To create an optimized build for deployment (like hosting on Oracle Cloud, Vercel, or Netlify):

```bash
npm run build
```
This will generate a `dist` folder containing the minified, production-ready static files. You can serve this folder using any web server like Nginx or Apache.
